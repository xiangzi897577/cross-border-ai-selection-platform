# Product Overview

## 产品定位

AI跨境选品分析平台是一个面向跨境轻小件卖家和运营团队的 AI 选品分析工具。它将商品基础信息、采购成本、平台费用、物流成本、市场表现、竞争强度、风险因素和 AI 建议整合到同一套决策流程中，帮助用户更系统地判断候选商品是否值得继续跟进。

当前版本以手机支架作为种子品类和演示数据，用于验证轻小件跨境选品分析流程。平台定位不局限于手机支架，后续可以扩展到桌面收纳、旅行配件、车载小配件、手机周边、厨房小工具、宠物小用品等更多跨境轻小件品类。

## 目标用户

- 需要快速筛选候选商品的跨境电商卖家。
- 需要评估商品利润、竞争和风险的选品人员。
- 需要维护商品候选池并进行持续观察的运营人员。
- 需要结合结构化指标和 AI 分析进行决策辅助的小团队。

## 解决的业务问题

轻小件跨境选品通常需要综合判断多个维度：

- 商品售价是否覆盖采购、物流和平台费用。
- 利润率是否有足够空间。
- 竞争强度是否过高。
- 评分、评论和销量是否反映稳定需求。
- 商品重量、体积和品类是否适合跨境物流。
- 候选商品是否值得进入下一轮供应链和市场验证。

本项目通过统一的数据模型和页面流程，把这些判断变成可视化、可筛选、可追踪的选品工作流。

## 核心工作流

1. 在 Dashboard 查看整体商品池质量，包括平均利润率、高潜力商品数、风险商品数和竞争情况。
2. 在商品列表中按关键词、类目、利润率和排序条件筛选候选商品。
3. 进入商品详情页查看利润、成本、风险、推荐评分和推荐理由。
4. 将值得跟进的商品加入候选池。
5. 在选品分析页查看高潜力、高风险、低竞争高利润等分组。
6. 使用 AI 对话或商品报告辅助判断下一步跟进方向。

## 决策信号

平台当前使用的主要选品信号包括：

- 采购成本
- Amazon 预估售价
- 平台手续费
- 物流成本
- 单件利润
- 利润率
- 月销量
- 评分
- 评论数量
- 竞争指数
- 风险因素
- 推荐评分
- 候选池状态

这些信号既用于页面展示，也用于 Dashboard 统计、风险分组、推荐排序和 AI 上下文。

## AI 辅助能力

AI 功能用于补充结构化指标难以直接表达的分析维度，例如：

- 对候选商品进行选品建议说明。
- 根据商品详情生成分析报告。
- 结合候选池上下文回答多轮问题。
- 对利润、竞争、物流和风险进行综合解释。

AI 只作为决策辅助，不替代真实市场调研、供应商沟通和平台规则校验。

## 当前版本限制

当前版本是后续迭代基线，仍有一些明确限制：

- 商品池规模有限，当前以手机支架种子品类为主，主要用于验证产品流程和指标设计。
- 1688 与 Amazon 数据尚未通过真实 API 自动同步。
- 当前商品主数据已迁移至 Supabase `products` 表，`server/data/products.json` 主要作为种子数据、备份数据和本地导入来源。
- AI 分析依赖后端提供的上下文质量，不能保证完全准确。
- 匿名 `client_id` 适合轻量候选池管理，不等同于完整用户系统。

## 后续扩展方向

后续产品路线重点围绕真实数据源、品类扩展和 AI 决策质量展开：

- 接入 1688 和 Amazon 等真实数据源，提升货源与市场数据时效性。
- 从手机支架种子品类扩展到更多轻小件跨境候选商品。
- 完善数据质量校验、刷新机制和异常兜底。
- 让 AI 建议更稳定地引用利润、竞争、风险和物流等结构化指标。
